Text-Mining-Fundamentals-Slides-
This repo is part of my teaching in NLP and text analytics.
It includes slides that give a general idea of how to start and understand the basics of text mining.
The slides highlight the main concepts and key steps in text analysis, including:

1) Understanding data types
2) Text preprocessing
3) Feature extraction
4) Topic modeling
5) Text classification
6) Sentiment analysis
7) Text summarization
8) Model evaluation and visualization
